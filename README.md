# 🖥️ Portfolio Template

A clean, responsive personal portfolio website template designed for UI/UX designers and developers.

## 🚀 Live Demo

> https://web-portfolio-ui.netlify.app/

## 📸 Key Sections

This template includes the following essential sections:

- **Hero**: An introductory section featuring a profile photo and a prominent Call-to-Action (CTA) button.
- **Worked With**: Showcases logos of companies or clients the user has collaborated with.
- **Case Studies**: Highlights key projects with detailed descriptions and relevant tags.
- **Testimonials**: Displays a grid of client reviews and endorsements.
- **Recent Work**: A scrollable slider presenting the latest projects.
- **Contact**: A straightforward form for visitors to get in touch.
- **Footer**: Contains essential links and copyright information.

## 🛠️ Technology Stack

The project is built using the following technologies:

- **HTML5**: For structuring the web content.
- **CSS3**: Utilizes custom properties (CSS variables) for flexible styling.
- **Bootstrap 5.3**: A robust framework for responsive design.
- **Vanilla JavaScript**: For interactive elements and dynamic content.
- **Google Fonts**: Incorporates 'Raleway' and 'IBM Plex Mono' for typography.

## ⚙️ Getting Started

To set up and run this project locally, follow these steps:

1.  **Clone the repository**:

    ```bash
    git clone https://github.com/MoSalem149/portfolio-template.git
    cd portfolio-template
    ```

2.  **Open in browser**: Navigate to the project directory and open `index.html` in your preferred web browser.

3.  **Customize content**: Replace all placeholder content, including your name, biography, images, and links, with your personal details.

## ✏️ Customization

Easily personalize the template to match your brand and content:

-   **Colors**: Update the primary and secondary colors by modifying the CSS variables defined in the `:root` selector within `style.css`.
-   **Images**: Replace the default images located in the `assets/images/` directory with your own project visuals and profile pictures.
-   **Social Links**: Add your professional social media links (e.g., LinkedIn, Behance, Twitter) to the navigation bar.
